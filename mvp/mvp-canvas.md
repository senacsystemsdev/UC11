# MVP Canvas — Controle de Validade com IA (Atacadão)

**UC11:** Gerir Projetos de Tecnologia da Informação  
**Equipe:** William, Alaide, Ed

---

| Seção | Descrição |
|-------|-----------|
| **Problema** | Perdas no varejo de R$ 42,1 bi/ano; 35% por vencimento; TAC MPDFT vigente; fiscalização do MP-BA; falha na execução do controle de validade do ERP |
| **Solução** | IA preditiva que cruza validade com velocidade de venda, sugere ações (desconto, realocação, descarte) e detecta anomalias em perdas |
| **Métrica-Chave** | Redução de produtos vencidos (meta: zero), redução de perdas totais (15%), tempo de detecção (-70%) |
| **Proposta de Valor** | Reduzir perdas, evitar multas e danos à reputação, automatizar detecção de anomalias em tempo real |
| **Público-Alvo** | Gerentes de loja, operadores de estoque, coordenadores de operações, analistas de prevenção de perdas, diretoria |
| **Canais** | Dashboard web, notificações push, relatórios mensais, API de integração com ERP |
| **Vantagem Competitiva** | Modelo ML treinado para o varejo atacadista; integração nativa com TOTVS Consinco/RMS; alertas em tempo real com baixa taxa de falso positivo |
| **Custos** | Infraestrutura cloud (servidores, banco), desenvolvimento (React + Node + Python), manutenção do modelo ML |
| **Receita** | Redução de perdas (ROI direto), redução de multas/indenizações, eficiência operacional |
| **Riscos** | Qualidade dos dados do ERP; adesão dos gerentes ao sistema; integração com legado; falsos positivos gerarem desconfiança |
